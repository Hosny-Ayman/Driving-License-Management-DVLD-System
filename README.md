#  Driving License Management System (DVLD)

A full desktop application that simulates the workflow of a **Driving & Vehicle License Department**.

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- SQL Server
- 3-Tier Architecture

## Architecture
- Presentation Layer (Windows Forms)
- Business Layer
- Data Access Layer

## Features
- People Management
- Driver & License Management
- Local & International License Applications
- License Renewal & Replacement
- Test Scheduling & Results
- Users & Authentication System
- Detained License Handling

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Update the database connection string
4. Run the application
